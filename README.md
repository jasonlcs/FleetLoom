# DataWeave

致敬 [NimbleText](https://nimbletext.com/Live) 的純文字樣板轉換工具，純自用。

原作 NimbleText 是超好用的文字處理工具，這個是它的單檔 HTML 致敬版。

## 說明

- 單檔 HTML，零依賴，打開即用
- DATA → FILTER（WHERE）→ PATTERN → RESULTS
- 支援 `<% JS %>` 內嵌表達式、正則分隔符、RFC 4180 CSV 解析
- 三語介面（中 / EN / 日）
- 內建日期函式庫、snippet 儲存（localStorage）
- 匯出 / 匯入 JSON 專案檔

## 使用

瀏覽器直接開啟 `dataweave.html` 即可。
